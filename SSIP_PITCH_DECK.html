1. First Offline Autonomous Pentest AI

Every competitor (PentestGPT, HackerGPT, Pentera) requires cloud/internet. FRIDAY runs a local 7B AI on a portable mini-PC. No internet needed. No data ever leaves the device.

How It Works

A Real-World Example: "Audit My Office Network"

What Happens Automatically

  • Step 1: User speaks: "FRIDAY, audit my office network"
  • Step 2: AI runs Nmap → discovers 12 devices, 47 open ports
  • Step 3: AI spots Apache 2.4.49 on port 80 → queries CVE database → finds CVE-2021-41773
  • Step 4: AI runs Nikto on web ports → finds SQL injection parameter
  • Step 5: AI runs SQLMap on that parameter → confirms vulnerability
  • Step 6: AI scans WiFi using adapter → checks for weak WPA2 passwords
  • Step 7: AI generates full PDF report with findings + fix recommendations

Key Technical Components

  • Local AI: Qwen2.5-Coder 7B model running via Ollama — no cloud needed
  • RAG Knowledge: Embedded HackTricks + OWASP guide for expert-level decisions
  • Tool Registry: Auto-detects all 600+ installed Kali/BlackArch tools at boot
  • Safety Layer: Every AI command validated before execution — blocked if dangerous
  • Target Model: SQLite knowledge graph stores all findings persistently
  • Anonymity: Tor + Proxychains + MAC spoofing enabled by default
Total time: 30 minutes (vs 2-3 days for manual pentest)
